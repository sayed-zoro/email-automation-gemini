Email Automation using Gemini (Google GenAI)

This project automates the generation of professional emails using Google Gemini (Generative AI).
It allows users to create well-structured, context-aware emails for use cases such as outreach, follow-ups, notifications, and customer communication.

Features

AI-powered email generation using Google Gemini

Prompt-based email creation

Streamlit user interface

SMTP support for sending emails

Clean and modular Python code

Tech Stack

Python

Google Gemini (Generative AI)

Streamlit

SMTP (Gmail or similar)

How It Works

The user provides an email context or prompt

The application sends the prompt to Google Gemini

Gemini generates a professional email response

The email can be reviewed and sent using SMTP
